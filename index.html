pyRevit Extension

Custom pyRevit tab deployed company-wide at NIRAS with 40+ tools built in Python using the Revit API. Designed to standardize workflows, reduce repetitive tasks, and enforce quality across all BIM projects.

  • Model Cleanup — automated purging of unused families, views, and line styles; bulk health checks
  • QA Checks — 2D element audits, dimension and text validation, in-place family detection, view filter overview and cross-model copy
  • Reinforcement Tools — rebar location tagging, partition splitting, unobscured visibility cleanup, and RFT detailing automation
  • Multi-Export — batch export to IFC, NWC, PDF, and DWG with configurable settings per view/sheet set
  • Element Picker — advanced selection and filtering with multi-criteria search across categories, parameters, and worksets
  • Model Checker — comprehensive audit generating reports on naming, parameters, warnings, and compliance
  • Productivity — coordinate setting, wall/floor type creation, parameter concatenation, sheet renumbering, browser reorganization
  • Help & Standards — integrated links to ACC, drawing standards, MOLIO, naming conventions, templates, and changelog/feedback
